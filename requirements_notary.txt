# via the ./engine symlink) runtime pins — install them into the engine's uv venv:

#   uv pip install --python engine/bonsai/.venv/bin/python -r requirements_notary.txt
#
# The on-chain software (chain_c) is C — built separately with cmake, NOT a pip dependency.
# The self-managed BSV HD wallet is optional (only if you fund/sign in Python instead of letting
# chain_c hold keys):  uv pip install -r requirements_wallet.txt
# Tests need pytest:    uv pip install -r requirements_test.txt
